transition-duration:필수 속성으로 트랜지션의 작동시간을 지정할 때 사용하는 속성임 지정시 시간단위 값을 지정해야 함

선택자 { transition-duration:2s;}



transition-delay:선택속성으로 트랜지션의 시작 전 지연시간을 의미하며 지정한 시간단위 값만큼 기다린 후 작동함

선택자 { transition-delay:2s;}

transition-property:트랜지션을 적용할 css 속성을 지정할 수 있으며 여러개의 css 속성을 지정한 경우 transition-duration이나 transition-delay 등을 달리 적용할 수 있음

선탯자 {transition-duration:2s; transition-delay:1s; transition-property:color, background, height, border }



transition-timing-function : 트랜지션의 가속 및 감속 효과를 지정

ease | ease-in | ease-out | ease-in-out | linear | cubic-bezier()

선택자 {transition-duration:4s; transition-timing-function:ease;}
